BOM(Browser Object Model)

navigator객체

웹 페이지를 실행하고 있는 브라우저에 대한 정보를 갖고 있는 객체

screen객체

웹 브라우저 화면이 아닌 모니터 화면의 속성을 갖는 객체

location객체

브라우저 주소 표시줄과 관련된 객체

location.href

페이지 이동

reload()

페이지를 새로고침하는 메소드

assign()과 replace()

두 메소드 모두 location자원을 제공한 url의 자원으로 재지정하는 메소드 = 페이지 이동
reaplce()는 assgin()과 다르게 history객체에 현재 resource를 남기지 않음

history객체

브라우저 방문 기록과 관련된 객체
